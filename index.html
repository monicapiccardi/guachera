📖 Cómo configurar Google Sheets

Paso 1 — Crear una planilla nueva

Abrí sheets.google.com y creá una hoja en blanco con el nombre que quieras.

Paso 2 — Abrir Apps Script

Dentro de la planilla: menú Extensiones → Apps Script.

Paso 3 — Pegar el código

Borrá todo lo que hay en el editor y pegá el código que copiás con el botón de abajo.

Paso 4 — Publicar como aplicación web

Clic en Implementar → Nueva implementación.
Tipo: Aplicación web.
Ejecutar como: Yo.
Quién tiene acceso: Cualquier usuario.
Copiá la URL generada y pegala aquí arriba.
